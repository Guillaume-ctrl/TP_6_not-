\doxysubsection{Detailed Description}
Represents a dice that can be rolled to generate random numbers. 

The \doxylink{class_dice}{Dice} class provides functionality to simulate a dice roll with a specified number of sides. It uses a Mersenne Twister random number generator to produce uniformly distributed random numbers.
