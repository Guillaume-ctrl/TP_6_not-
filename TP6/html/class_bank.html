Detailed Description

template<typename T>
class Bank< T >

A generic class to manage a bank's money operations.

The Bank class provides methods to add, deduct, and retrieve the current amount of money. It is templated to allow flexibility with different numeric types.

Template Parameters
TThe type used for money (e.g., int, float, unsigned int).
